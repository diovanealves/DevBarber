## 🚀 Instalando DevBarber

Para instalar o DevBarber, siga estas etapas:

Linux e macOS:
```
npm install
    ou
yarn install
```

Windows:
```
npm install
    ou
yarn install
```

## ☕ Usando DevBarber

Para usar DevBarber, siga estas etapas:

```
npm run dev
    ou
yarn dev
```
